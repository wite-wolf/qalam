# قلم - السبورة البيضاء الذكية

تطبيق سبورة بيضاء ذكية للرسم والكتابة مع أدوات احترافية.

## 🚀 المميزات
- أدوات رسم متعددة: قلم، ممحاة، هايلايتر، أشكال، أسهم، نصوص
- تراجع وإعادة (Undo/Redo)
- تكبير وتصغير وتحريك (Zoom/Pan)
- حفظ واسترجاع المشاريع مع صور مصغرة
- تصدير بصيغ: PNG, JPG, .qalam (قابل للتعديل)
- اختصارات لوحة المفاتيح
- واجهة عربية 100%
- متوافق مع Windows, Android, Web (PWA)

## 🛠 بناء من المصدر
```bash
# Windows
npm install
npm run pack:win

# Android (يتطلب Android Studio + Java JDK)
npm run build:www
npx cap sync android
npx cap open android
```
